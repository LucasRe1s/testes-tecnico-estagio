1 - Diferenciar as camadas 2 e 3 do modelo OSI, e indicar os protocolos utilizados para endereçamento
 nestas camadas.
A camada 2 (Enlace de dados), se preocupa quando se refere às tecnologias usadas para conectar computadores em uma mesma rede
aonde a camada fisica já existe. A camada 2 geralmente é dividida em duas subcamadas: a camada Media Access 
Control (MAC) e a camada Logical Link Control (LLC). Os protocolos usados são o Ethernet, Wi-Fi.

Já a camada 3 (Rede) se preocupa com os conceitos de roteamento, encaminhamento, endereçamento em uma rede 
dispersa ou em varias redes conectadas. A camada de rede também pode gerenciar o controle de fluxo. Na 
Internet, o Internet Protocol v4 (IPv4) e o IPv6 são usados ​​como os principais protocolos da camada de rede. Os
protocolos usados são IP Internet Protocol, ICMP Internet Control Message Protocol, ARP Address Resolution Protocol, 
RIP Routing Information Protocol.

3 - O que seria um projeto OpenSource? Como empresas podem adotar tais tecnologias e o que isso acarreta?
R: Um projeto OpenSource, é um software de codigo aberto, e significa que qualqyer um pode ver, usar, 
modificar e distribuir o projeto por qualquer motivo. Empresas podem adotar usando desde sistemas operacionais
como linux, usando servidores web como Apache, ou usando banco de dados como PostgresSQL, MySQL, Redis. Isso acarreta
em inovação pois vão ter pessoas de fora do projeto com ideias diferentes, redução de custos com licenças, customização
para melhor resolução do que os agradam seja uma interface padronizada ou até mesmo alguma modificação em um sistema
pois está sujeito a adaptação.
